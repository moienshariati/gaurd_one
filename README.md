# ViperOne Guard

Static React prototype for a mobile security guard shift workflow.

## Local Preview

```bash
npm run dev
```

The local preview runs at `http://localhost:4173`.

## Vercel

This project deploys as a static site. Vercel should use the repository root as the output directory and does not need a build command.

The main app is served from `/`. The design overview is available at `/canvas`.
